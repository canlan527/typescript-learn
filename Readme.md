# Typescript 重构 axios
使用 typescript 重构 axios
## 启动后端 express 服务
```bash
pnpm run dev
```
## express 文件位置
```
src/server/index.js
```
## 启动 Webpack 脚手架 运行项目代码
```bash
pnpm run serve
```
## 运行测试实例
根据 `src/index.ts` 文件导入的不同文件运行不同示例
```javascript
import './example/01_simple'
import './example/02_simple'
import './example/03_simple'
```
## 添加示例文件
根据`src/example`下不同文件创建不同axios示例，启动程序。
